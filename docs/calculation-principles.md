# 火炬之光无限伤害计算原理与公式

## 1. 伤害计算流程

伤害计算是一个多步骤的过程，从基础伤害开始，经过各种加成和修正，最终得到最终的DPS（每秒伤害）。

### 计算流程概览
1. **基础伤害计算**：根据技能类型（攻击/法术）计算基础伤害
2. **伤害转换**：处理伤害类型转换（如物理转元素）
3. **伤害放大**：应用增加伤害和更多伤害加成
4. **暴击计算**：计算暴击率和期望暴击伤害
5. **双倍伤害计算**：处理双倍伤害效果
6. **攻击速度计算**：计算每秒攻击次数
7. **DPS计算**：综合所有因素计算最终DPS
8. **异常状态伤害计算**：计算DOT（持续伤害）

## 2. 基础伤害计算

### 2.1 攻击技能基础伤害

**公式**：
```
基础伤害 = 武器面板伤害 × 技能伤害百分比 + 附加伤害
```

**参数说明**：
- **武器面板伤害**：武器的基础伤害范围（最小值和最大值）
- **技能伤害百分比**：技能本身的伤害系数，随技能等级增加
- **附加伤害**：来自装备、天赋等的附加点伤，受技能伤害效用影响

**计算示例**：
- 武器面板伤害：100-200物理伤害
- 技能伤害百分比：200%
- 附加伤害：50-100物理伤害
- 技能伤害效用：1.0
- 计算结果：(100-200) × 2 + (50-100) = 250-500物理伤害

### 2.2 法术技能基础伤害

**公式**：
```
基础伤害 = 法术基础点伤 × 技能伤害百分比 + 附加伤害
```

**参数说明**：
- **法术基础点伤**：法术技能的基础伤害范围
- **技能伤害百分比**：技能本身的伤害系数，随技能等级增加
- **附加伤害**：来自装备、天赋等的附加点伤，不受技能伤害效用影响

**计算示例**：
- 法术基础点伤：50-100火焰伤害
- 技能伤害百分比：150%
- 附加伤害：25-50火焰伤害
- 计算结果：(50-100) × 1.5 + (25-50) = 100-200火焰伤害

## 3. 伤害转换

### 3.1 伤害类型转换

**公式**：
```
转换后伤害 = 原伤害 × 转换百分比
剩余原伤害 = 原伤害 × (1 - 转换百分比)
```

**示例**：
- 100物理伤害，50%物理转火焰
- 转换后：50物理伤害 + 50火焰伤害

### 3.2 多次转换

**规则**：多次转换按顺序计算，后一次转换基于前一次转换的结果

**示例**：
- 100物理伤害
- 50%物理转火焰 → 50物理 + 50火焰
- 50%火焰转闪电 → 50物理 + 25火焰 + 25闪电

## 4. 伤害放大

### 4.1 增加伤害（Increased Damage）

**规则**：所有增加伤害的效果相加

**公式**：
```
增加伤害倍数 = 1 + (增加伤害1 + 增加伤害2 + ...) / 100
```

**示例**：
- 装备提供20%增加伤害
- 天赋提供30%增加伤害
- 技能提供10%增加伤害
- 计算结果：1 + (20 + 30 + 10) / 100 = 1.6倍

### 4.2 更多伤害（More Damage）

**规则**：所有更多伤害的效果相乘

**公式**：
```
更多伤害倍数 = (1 + 更多伤害1/100) × (1 + 更多伤害2/100) × ...
```

**示例**：
- 装备提供10%更多伤害
- 天赋提供15%更多伤害
- 技能提供5%更多伤害
- 计算结果：(1 + 0.1) × (1 + 0.15) × (1 + 0.05) = 1.32825倍

## 5. 暴击计算

### 5.1 暴击率

**公式**：
```
暴击率 = 基础暴击率 + 装备/天赋提供的暴击率
```

**说明**：
- 基础暴击率通常为5%
- 暴击率上限通常为95%

### 5.2 暴击伤害

**公式**：
```
暴击伤害 = 基础暴击伤害 + 装备/天赋提供的暴击伤害
```

**说明**：
- 基础暴击伤害通常为150%

### 5.3 期望暴击伤害

**公式**：
```
期望暴击伤害 = (1 - 暴击率) × 1 + 暴击率 × 暴击伤害
```

**示例**：
- 暴击率：50%
- 暴击伤害：200%
- 计算结果：(1 - 0.5) × 1 + 0.5 × 2 = 1.5倍

## 6. 双倍伤害计算

### 6.1 双倍伤害期望

**公式**：
```
期望双倍伤害 = (1 × (1 - 双倍伤害概率)) + (2 × 双倍伤害概率)
```

**示例**：
- 双倍伤害概率：10%
- 计算结果：(1 × 0.9) + (2 × 0.1) = 1.1倍

## 7. 攻击速度计算

### 7.1 攻击速度

**公式**：
```
攻击速度 = 基础攻击速度 × (1 + 攻击速度增加%)
```

**示例**：
- 基础攻击速度：1.0次/秒
- 攻击速度增加：50%
- 计算结果：1.0 × (1 + 0.5) = 1.5次/秒

## 8. DPS计算

### 8.1 最终DPS

**公式**：
```
DPS = 基础伤害 × 增加伤害倍数 × 更多伤害倍数 × 期望暴击伤害 × 期望双倍伤害 × 攻击速度
```

**示例**：
- 基础伤害：100
- 增加伤害倍数：1.5
- 更多伤害倍数：1.2
- 期望暴击伤害：1.3
- 期望双倍伤害：1.1
- 攻击速度：1.5次/秒
- 计算结果：100 × 1.5 × 1.2 × 1.3 × 1.1 × 1.5 = 400.95

### 8.2 总DPS

**公式**：
```
总DPS = 击中DPS + DOT DPS
```

**说明**：
- 击中DPS：直接击中造成的伤害
- DOT DPS：异常状态持续伤害

## 9. 异常状态伤害计算

### 9.1 DOT伤害

**公式**：
```
DOT DPS = 基础伤害 × 异常状态伤害系数 × 持续时间修正 × (1 - 敌人抗性)
```

**参数说明**：
- **异常状态伤害系数**：不同异常状态的伤害系数
- **持续时间修正**：持续时间对DOT伤害的影响
- **敌人抗性**：敌人对该伤害类型的抗性

### 9.2 常见异常状态

- **燃烧**：火焰伤害DOT
- **冰冻**：寒冷伤害DOT
- **感电**：闪电伤害DOT
- **腐蚀**：腐蚀伤害DOT

## 10. 防御计算

### 10.1 抗性计算

**公式**：
```
有效抗性 = 基础抗性 + 装备/天赋提供的抗性 - 敌人穿透
```

**说明**：
- 抗性上限通常为75%
- 抗性穿透会降低敌人的有效抗性

### 10.2 护甲减伤

**公式**：
```
护甲减伤 = 护甲值 / (护甲值 + 10000)
```

**说明**：
- 护甲减伤上限通常为90%

## 11. 特殊机制

### 11.1 持续伤害加成

**规则**：持续伤害有专门的加成机制，与直接伤害分开计算

### 11.2 元素超载

**效果**：暴击时增加元素伤害，持续一定时间

### 11.3 物理点伤转化

**规则**：某些技能和装备可以将物理点伤转化为元素点伤

## 12. 计算示例

### 12.1 完整计算示例

**输入参数**：
- 武器面板：100-200物理伤害
- 技能伤害百分比：200%
- 附加伤害：50-100物理伤害
- 增加伤害：50%
- 更多伤害：20%
- 暴击率：50%
- 暴击伤害：200%
- 双倍伤害概率：10%
- 攻击速度：1.5次/秒
- 敌人物理抗性：20%

**计算过程**：
1. 基础伤害：(100-200) × 2 + (50-100) = 250-500物理伤害
2. 增加伤害倍数：1 + 50/100 = 1.5
3. 更多伤害倍数：1 + 20/100 = 1.2
4. 期望暴击伤害：(1-0.5) × 1 + 0.5 × 2 = 1.5
5. 期望双倍伤害：(1-0.1) × 1 + 0.1 × 2 = 1.1
6. 攻击速度：1.5次/秒
7. 击中DPS：(250-500) × 1.5 × 1.2 × 1.5 × 1.1 × 1.5 = 826.875-1653.75
8. 考虑敌人抗性：(826.875-1653.75) × (1-0.2) = 661.5-1323

**最终结果**：
- 击中DPS：661.5-1323
- 总DPS：取决于DOT伤害

## 14. 注意事项

1. **版本差异**：不同游戏版本的计算机制可能有所不同
2. **特殊效果**：某些特殊效果可能有独特的计算方式
3. **精度问题**：游戏内计算可能存在取整或其他精度处理
4. **测试验证**：计算结果应与游戏内实际伤害进行对比验证

## 15. 常见问题解答

### 15.1 为什么计算结果与游戏内不符？
- 可能存在未考虑的特殊效果
- 装备属性输入错误
- 技能等级或配置不正确

### 15.2 如何提高计算精度？
- 仔细输入所有装备和天赋属性
- 考虑所有可能的伤害加成
- 验证特殊效果的计算方式

### 15.3 支持哪些伤害类型？
- 物理伤害
- 火焰伤害
- 寒冷伤害
- 闪电伤害
- 腐蚀伤害
