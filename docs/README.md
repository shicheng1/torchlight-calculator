# 火炬之光无限伤害计算器使用文档

## 1. 概述

火炬之光无限伤害计算器是一款专业的游戏角色伤害计算工具，旨在帮助玩家精确计算角色的伤害输出，优化装备和天赋选择，提升游戏体验。

### 主要功能
- 实时伤害计算和分析
- 装备、天赋、技能等多维度属性管理
- 伤害分解和详细数据展示
- 多种英雄和技能支持
- 导入/导出功能

## 2. 快速开始

### 2.1 系统要求
- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 互联网连接（首次加载需要下载资源）

### 2.2 基本操作
1. **选择英雄**：在顶部选择你要计算的英雄
2. **配置装备**：在装备面板中添加和编辑装备
3. **选择技能**：在技能面板中选择和配置技能
4. **调整天赋**：在天赋面板中分配天赋点
5. **查看结果**：在结果面板中查看详细的伤害计算结果

## 4. 伤害计算原理

### 4.1 基础伤害计算

**攻击技能**：
```
基础伤害 = 武器面板伤害 × 技能伤害百分比 + 附加伤害
```

**法术技能**：
```
基础伤害 = 法术基础点伤 × 技能伤害百分比 + 附加伤害
```

### 4.2 伤害放大计算

**增加伤害（Increased Damage）**：
- 所有增加伤害的效果相加
- 计算公式：`1 + (增加伤害1 + 增加伤害2 + ...) / 100`

**更多伤害（More Damage）**：
- 所有更多伤害的效果相乘
- 计算公式：`(1 + 更多伤害1/100) × (1 + 更多伤害2/100) × ...`

### 4.3 暴击计算

**暴击率**：
- 基础暴击率 + 装备/天赋提供的暴击率

**暴击伤害**：
- 基础暴击伤害（通常为150%） + 装备/天赋提供的暴击伤害

**期望暴击伤害**：
```
期望暴击伤害 = (1 - 暴击率) × 1 + 暴击率 × 暴击伤害
```

### 4.4 攻击速度计算

**攻击速度**：
- 基础攻击速度 × (1 + 攻击速度增加%)

### 4.5 DPS计算

**DPS（每秒伤害）**：
```
DPS = 基础伤害 × 增加伤害倍数 × 更多伤害倍数 × 期望暴击伤害 × 攻击速度
```

### 4.6 异常状态伤害

**异常状态DOT伤害**：
- 基于基础伤害和持续时间计算
- 考虑敌人抗性和各种加成

## 6. 常见问题

### 6.1 计算结果与游戏内不符
- 检查装备属性是否正确
- 确认技能等级和配置
- 验证天赋点分配

### 6.2 如何提高伤害
- 优先提升主要伤害类型的加成
- 平衡暴击率和暴击伤害
- 合理搭配装备和天赋

### 6.3 支持哪些游戏版本
- 当前支持火炬之光无限最新版本
- 随着游戏更新，计算器会定期更新
